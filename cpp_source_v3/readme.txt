This project includes C++ code that is useful for RL research. Files are distributed as follows:

build: Files for building the project in different IDEs

include: Third party code that needs to be in the include path (e.g., Eigen)

src: .cpp files. Subdirectories are included to help with organization.

header: .h and .hpp files. Subdirectories are included to help with organization.

install: Contains files that must be placed elsewhere on the machine
